1. Запускаем браузер Chrome
2. Заходим на сайт lostfilm
3. Открываем файл series.txt и считываем название сериала на английском языке (это важно)
4. Открываем файл rating.txt, при его отсутствии создаем такой файл
5. Ищем поисковую строку и вводим название сериала из текстового файла series.txt

6. Нажимаем на кнопку все результаты
7. Нажимаем на строчку названия сериала на английском языке (название считывается из файла series.txt)
8. Когда зашли на страницу сериала, ищем его рейтинг
9. Считываем рейтинг сериала и записываем в файл rating.txt "Рейтинг сериала + name + rating"
10. Закрываем браузер Chrome
